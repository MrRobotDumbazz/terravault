# Roadmap

## v1.0 — Hackathon MVP (current)
- [x] Core tokenization protocol (SPL Token-2022 + TransferHook)
- [x] USDC escrow vault with milestone-based release
- [x] Oracle-submitted construction proofs
- [x] 48-hour milestone dispute window
- [x] On-chain income distribution (rental + sale proceeds)
- [x] DAO dispute resolution (raise → evidence → admin resolve)
- [x] Developer blacklist and token account freeze
- [x] Emergency pause (circuit breaker)
- [x] Oracle key rotation with 48-hour timelock
- [x] Go oracle service (proof signer + REST API + on-chain listener)
- [x] React + Vite frontend

## v1.1 — Mainnet Alpha
- [ ] Mainnet deployment
- [ ] Security audit (milestone escrow + distribution math)
- [ ] Validator integration guide
- [ ] Performance benchmarks (TPS under load)
- [ ] Persona KYC production integration
- [ ] PostgreSQL schema migrations and indexer stability
- [ ] Developer onboarding documentation

## v2.0 — Full Protocol
- [ ] TypeScript SDK (`@terravault/sdk`)
- [ ] Multi-signature developer custody
- [ ] ZK-proofed property valuations (Pyth oracle integration)
- [ ] Conditional milestone execution (on-chain condition checks)
- [ ] Privacy layer for investor positions (ZK token balances)
- [ ] Investor dashboard with P&L tracking
- [ ] Bundle analytics for oracle submissions

## v3.0 — Ecosystem
- [ ] Jupiter integration for secondary market token liquidity
- [ ] Raydium / Orca liquidity pool bootstrapping
- [ ] Governance token and DAO treasury
- [ ] Decentralized oracle network (multiple oracle signers)
- [ ] Cross-chain bridge (Wormhole) for non-Solana investors
- [ ] Integration with on-chain title registries (pilot jurisdictions)
- [ ] Mobile app (React Native)

## Long-term Vision
TerraVault aims to become the canonical real estate investment primitive on Solana — a composable protocol that any DeFi application can plug into for property-backed yield, collateral, or structured products.
